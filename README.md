# Matrizes

Trabalho de Algoritmos em C — Matrizes

# Objetivo
Desenvolver programas em linguagem C utilizando matrizes, entrada de dados, laços de repetição e processamento de valores armazenados em estruturas bidimensionais.

# Questão 1 — Soma de Matrizes
Faça um programa em C que leia duas matrizes A e B, ambas de tamanho 3 x 3, e calcule uma terceira matriz C, em que cada posição seja a soma dos elementos correspondentes das duas matrizes.

Ou seja:

C[i][j] = A[i][j] + B[i][j];
Ao final, o programa deve exibir:

A matriz A;

A matriz B;

A matriz resultante C.

# Questão 2 — Maior Valor em uma Matriz
Faça um programa em C que leia uma matriz de inteiros de tamanho 4 x 4 e identifique o maior valor armazenado nela.

O programa deve exibir:

A matriz informada;

O maior valor encontrado;

A posição em que o maior valor está localizado, indicando linha e coluna.
